The row containing the letters (using col-sm) breaks first. This happens because col-sm specifies a fixed width for each letter, so as the screen size decreases, the letters may wrap onto the next line to fit. On the other hand, the city names (using col-md) adapt their width to the available space, allowing them to remain on the same line for longer before wrapping.
